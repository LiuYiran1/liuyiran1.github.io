# 一些技巧

[128. 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)

给一个未排序的整数数组 `nums` ，找出数字连续的最长序列的长度，要求时间复杂度为 `O(n)`

- 优化思路：
  - 对于数字 x，如果 x - 1 存在于集合中，说明 x 肯定不是某个连续序列的第一个数（它可以被合并到以 x-1 或更小的数开头的序列中）。直接跳过，不处理。
  - 对于数字 x，如果 x - 1 不存在于集合中，说明 x 是一个连续序列的开头。此时才开始向后匹配 x+1, x+2... 直到断裂。

- 代码：
  ```java
  class Solution {
      public int longestConsecutive(int[] nums) {
          Set<Integer> st = new HashSet<>();
          for (int num : nums) {
              st.add(num); // 把 nums 转成哈希集合
          }
  
          int ans = 0;
          for (int x : st) { // 遍历哈希集合
              if (st.contains(x - 1)) { // 如果 x 不是序列的起点，直接跳过
                  continue;
              }
              // x 是序列的起点
              int y = x + 1;
              while (st.contains(y)) { // 不断查找下一个数是否在哈希集合中
                  y++;
              }
              // 循环结束后，y-1 是最后一个在哈希集合中的数
              ans = Math.max(ans, y - x); // 从 x 到 y-1 一共 y-x 个数
          }
          return ans;
      }
  }
  ```

  - 同时，如果找到一个序列比哈希集合的一半还大就可以直接返回
